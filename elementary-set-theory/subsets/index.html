Subsets

Subsets

Given a set $X$, a subset of $X$ is a set that contains some, but (often) not all the elements of $X$. For example, the set $E$ of even natural numbers is a subset of the set $\NN$ of all natural numbers. When viewing the set $E$ in isolation, however, it's not clear that it's a subset of $\NN$ or which elements of $\NN$ it contains. In fact, the set $E$ seems identical to $\NN$ as there is a bijection $n \mapsto \frac{n}{2}$ between the two sets. We need some way to realize the elements of $E$ as elements of $\NN$. This is done with an injection $m: E \hookrightarrow \NN$ serving as an inclusion of $E$ in $\NN$, mapping every even number $n \in E$ to its corresponding natural number in $\NN$, allowing us to compare elements of $E$ with elements of $\NN$. We can thus define a subset of $X$ to be a pair $(A,m)$ consisting of a set $A$ and an inclusion $m: A \hookrightarrow X$. We write $A \subset X$ (sometimes $A \subseteq X$) to denote $A$ is a subset of $X$, the inclusion $m$ is omitted in the notation as it's often clear from context. In this chapter, I'll often write $(A,m) \subset X$ instead. For $x \in X$, we write $x \in A$ for "$x$ is in the subset $A$" or, formally, $\exists a \in A. m(a) = x$, and we write $x \notin A$ for "$x$ is not in the subset $A$", which is simply the negation of $x \in A$.

We can also identify the set of even numbers with $\NN$ itself and the inclusion $n \mapsto 2n$. Or we could define a different inclusion, such as

\[\left\{\begin{array}{lll} 2n & \mapsto & 4n+2 \\ 2n+1 & \mapsto & 4n \end{array}\right.\]

which swaps each consecutive pair of even numbers. Besides these all being different sets and/or different inclusion, they all still represent the same subset of $\NN$: the set of even numbers. The definition of subsets we have is thus incomplete: it only says what a subset is, but not yet when two subsets are equal. The definition of the identity relation on subsets is as follows:

Given a set $X$ and subsets $(A,m) \subset X$ and $(B,n) \subset X$, we have that $(A,m)$ and $(B,n)$ are identical (or equal) if there is a bijection $f: A \leftrightarrow B$ for which $n = m \circ f$.

For example, the latter two representations of the set of even numbers are identical because of the bijection

\[\left\{\begin{array}{lll} 2n & \mapsto & 2n+1 \\ 2n+1 & \mapsto & 2n \end{array}\right.\]

It can be shown that, if two subsets are equal, then they're equal in a unique way. In other words, given $X$, $(A,m) \subset X$, $(B,n) \subset X$, $f: A \leftrightarrow B$ and $g: A \leftrightarrow B$ for which $n = m \circ f$ and $n = m \circ g$, we have that $f = g$. We use function extensionality to prove this: given $a \in A$, we have that $m(f(a)) = n(a) = m(g(a))$, and therefore $m(f(a)) = m(g(a))$. Since $m$ is an injection, we have that $f(a) = g(a)$, which is exactly what we wanted to show.

It's also possible for sets to be subsets of other subsets. For example, the set of $2$ and $6$ as a subset of the set of natural numbers, is a subset of the set of even numbers: every element of $\{2,6\}$, realized as an element of $\NN$, is also an element of $E$. This is equivalent to saying that there is an inclusion $i$ of $\{2,6\}$ in $E$ that "respects" the inclusions $m: E \hookrightarrow \NN$ and $n: \{2,6\} \hookrightarrow \NN$, i.e. for which $m \circ i = n$. For $(A,m) \subset X$ and $(B,n) \subset X$, we can thus define the proposition $A \subset B$ to mean that there exists an injection $i: A \hookrightarrow B$ for which $m \circ i = n$. The reader may verify that such an injection $i$, if it exists, must be unique. From this definition, we have that $(A,m)$ and $(B,n)$ are identical as subsets of $X$ exactly when both $A \subset B$ and $B \subset A$.

Predicates

If we have some rule for which elements belong to a subset and which don't, it's often difficult or sometimes just tedious to find a set $A$ and inclusion $m$ that selects exactly those elements that satisfy the given rule. For example, in the last section, if we don't have the set of even numbers $E$, we have to first identify that there are as many even numbers as natural numbers and then define an inclusion $\NN \hookrightarrow \NN$ that selects exactly the even numbers, such as $n \mapsto 2n$. Though it's not that difficult in this specific example, you can see it gets more difficult if you try to define the set of real-valued continuous functions as a subset of the set of all real-valued functions, which is still somewhat do-able with an inclusion from $\RR$, or the set of perfect numbers as a subset of $\NN$, for which we don't know how many there are and thus don't know which set we should choose as the domain of our inclusion. Because of this, there is an axiom in set theory, called the axiom of separation, which (roughly) states that every rule that selects elements of a set directly corresponds to some subset of that set.

We use $\Omega$ to denote the set of truth-values $\{\top, \bot\}$ ($\top$ = true, $\bot$ = false). A "rule" is then a function $P: X \to \Omega$ that maps each element $x \in X$ to either $\top$ or $\bot$. We call such a function a predicate on $X$. Since elements of $\Omega$ are truth-values, we can use the function value $P(x)$ as a proposition in a longer formula, as in $\forall x \in X. P(x)$, which states that $P(x)$ is true for all $x \in X$. The axiom of separation then states that, given any predicate $P: X \to \Omega$, there is a subset $(A,m) \subset X$ for which, for all $x \in X$, $x \in A$ if and only if $P(x)$. We use the notation $\{x \in X \mid P(x)\}$, called set comprehension notation or set builder notation, to denote this subset. This axiom allows us to define the set $\{n \in \NN \mid n \textrm{ is prefect}\}$ of perfect numbers without knowing how many there are in advance. Sometimes we use a colon $:$ instead of a pipe $\mid$ to separate the element and the predicate in a set comprehension expression.

In future chapters, we'll often identify predicates $P(x)$ with their corresponding subset $\{x \in X \mid P(x)\}$ and vice-versa. For now though, we'll keep them separate. It might make more sense to view the predicates on $X$ as the actual subsets, as two predicates $P,Q$ are equal as functions exactly when their corresponding subsets $\{x \in X \mid P(x)\}$ and $\{x \in X \mid Q(x)\}$ are equal as subsets.

A predicate $R: A \times B \to \Omega$ on a Cartesian product $A \times B$ is called a relation between $A$ and $B$. For $(a,b) \in A \times B$, we often write $a R b$ for $R(a,b)$. We'll taker a closer look at relations in the chapter about quotients.

Unions, Intersections, Complements

Three basic operations on sets are (as the title of this section says) unions, intersections and complements. Given two subsets $A,B \subset X$, we define their union as follows:

\[A \cup B = \{x \in X \mid x \in A \vee x \in B\}\]

Here, $\vee$ is the "or" symbol. A proposition of the form $P \vee Q$ is true if at least one of $P$ or $Q$ is true. The union of $A$ and $B$ thus contains all elements that are in either $A$ or in $B$ or both. In this picture, the two circles represent the two subsets $A$ and $B$ and the blue area is their union:

Unions of sets satisfy a few basic identities:

  • Union is associative: $(A \cup B) \cup C = A \cup (B \cup C)$;
  • Union is commutative: $A \cup B = B \cup A$;
  • The empty set is neutral: $A \cup \varnothing = A = \varnothing \cup A$

$\varnothing$ denotes the empty subset of $X$. All of these identities should be fairly straightforeward: changing the order in which you take unions doesn't change what elements are in the union, and adding nothing to a subset doesn't change that subset.

For subsets $A,B \subset X$, we say $A$ is a subset of $B$ (written $A \subset B$) if $\forall x \in X. x \in A \implies x \in B$. With this definition, we also have:

  • $A \subset A \cup B$;
  • $B \subset A \cup B$.

Furthermore, $A \cup B$ is a subset of any set $C$ satisfying the above two properties.

Given two subsets $A,B \subset X$, we define their intersection as follows:

\[A \cap B = \{x \in X \mid x \in A \wedge x \in B\}\]

Here, $\wedge$ is the "and" symbol. A proposition of the form $P \wedge Q$ is true if both $P$ and $Q$ are true. Thus, the intersection of $A$ and $B$ contains all elements that are in both $A$ and $B$. The shaded area in this picture represents the intersection of $A$ and $B$:

Just like unions, intersections satisfy a few basic properties:

  • Intersection is associative: $(A \cap B) \cap C = A \cap (B \cap C)$;
  • Intersection is commutative: $A \cap B = B \cap A$;
  • The whole set $X$ is neutral: $A \cap X = A = X \cap A$.

$X \subset X$ is the set of all elements of $X$. We also have some identities for how unions and intersections interact:

  • Unions distribute over intersections: $A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$;
  • Intersections distribute over unions: $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$.

It shouldn't be too hard to verify these identities for yourself.

We also have the following:

  • $A \cap B \subset A$;
  • $A \cap B \subset B$.

In fact, any subset $C$ with the above two properties is a subset of $A$.

Given a subset $A \subset X$, we define its complement $A^c$ as follows:

\[A^c = \{x \in X \mid x \notin A\}\]

In other words, it's the set of all elements of $X$ that are not in $A$. Unlike with unions and intersections, the ambient set $X$ now matters: if $X$ was a subset of a larger set $Y$ with an inclusion that respects those of $A$ and $B$, it didn't matter if we take the union of intersection of $A$ and $B$ in $X$ or in $Y$, but that does matter for complements. The blue area in this picture represents the complement of $A$ in $X$:

We have the following identities for how complements "distribute" over unions and intersections:

  • $(A \cup B)^c = A^c \cap B^c$;
  • $(A \cap B)^c = A^c \cup B^c$.

These identities are called De Morgan laws. These identities should make sense intuitively: if something is in not in either $A$ or $B$, then it is neither in $A$ nor $B$, and if something is not in both $A$ and $B$, it's either not in $A$ or not in $B$. We also have the following identity, showing taking complements is an involution:

  • $A^{cc} = A$.

$A^c$ is the unique subset of $X$ that satisfies the following two identities:

  • $A^c \cup A = X$;
  • $A^c \cap A = \varnothing$.

If it's not clear what set the subset $A$ is should be considered the subset of when taking a complement, we write $X \setminus A$ for $A^c$ to clarify this. We can also define $B \setminus A$ for any two subsets $A,B \subset X$:

\[B \setminus A = B \cap A^c = \{x \in X \mid x \in B \wedge x \notin A\}\]

The subset $B \setminus A$ is called the relative complement of $A$ relative to $B$. It looks like this:

We can also define another operation $A \bigtriangleup B$, called the symmetric difference, as follows:

\[A \bigtriangleup B = (B \setminus A) \cup (A \setminus B) = (A \cup B) \setminus (A \cap B)\]

This consists of all elements of $X$ that are in either $A$ or $B$, but not both. It can be used as a measure for how different two subsets are. For example, if $A \bigtriangleup B = \varnothing$, then $A$ and $B$ are the same subset.

Function Restrictions

Let $f: X \to Y$ be a function and let $(A,m) \subset X$ be a subset. The restriction of $f$ to $A$ (along $m$) is commonly denoted $f|A$, $f|_A$, $f \upharpoonright A$ or $f{\upharpoonright_A}$. We'll use the notation $f|_A$. The restriction $f|_A$ is a function from $A$ to $Y$ representing the restriction of the function $f$ to the domain $A$. I.e. for $a \in A$, we have $f|_A(a) = f(a)$. Formally, it is defined by:

\[f|_A = f \circ m\]

For $(A,m) \subset X$, $(B,n) \subset X$ and $f: X \to Y$, we have the following identities:

  • $f|_X = f$, where $X$ (with the inclusion $\id_X$) is the subset of all elements of $X$;
  • $f|_\varnothing$ is the empty function to $Y$;
  • Given that $A \subset B$ in $X$, witnessed by the inclusion $i: A \hookrightarrow B$, we have $(f|_B)|_A = f|_A$ (the first restriction to $A$ is along $i$, the second is along $m$);
  • $\id_X|_A = m$.

Given subsets $A \subset X$ and $B \subset X$ and functions $f: A \to Y$ and $g: B \to Y$, if $f|_{A \cap B} = g|_{A \cap B}$ (i.e. their values agree on their common domain), we can glue $f$ and $g$ together into a new function $f \cup g$. The union $f \cup g$ is a function from $A \cup B$ to $Y$ with the following two properties:

  • $(f \cup g)|_A = f$;
  • $(f \cup g)|_B = g$.

This function can be defined by cases as follows: $(f \cup g)(x) = f(x)$ if $x \in A \setminus B$, $(f \cup g)(x) = g(x)$ if $x \in B \setminus A$, and (either definition works) $(f \cup g)(x) = f(x)$ if $x \in A \cap B$ or, equivalently, $(f \cup g)(x) = g(x)$ if $x \in A \cap B$. A function from $A \cup B$ to $Y$ with the above two properties, if it exists, must be unique: suppose $h$ and $h'$ are such functions. We prove $h = h'$ using function extensionality. For $x \in A \cup B$, we have $x \in A$ or $x \in B$ (by definition of union). In the first case, i.e. if $x \in A$, we have that $h(x) = h|_A(x) = f(x) = h'|_A(x) = h'(x)$. In the second case, $h(x) = h|_B(x) = g(x) = h'|_B(x) = h'(x)$. In either case, $h(x) = h'(x)$.

Given an operator $f: X^n \to X$ ($X^n = X \times \cdots \times X$ with $n$ copies of $X$), we say a subset $A \subset X$ is closed under the operator $f$ if for all $a_1, \ldots, a_n \in A$, we have that $f(a_1,\ldots,a_n) \in A$ as well. In other words, the restriction $f|_{A^n}$ still makes sense as a function when we change the domain from $X$ to $A$. In yet other words, $\im(f) \subset A$. For example, consider the operation of addition on the natural numbers ${+}: \NN \times \NN \to \NN$. The set of even numbers is closed under this operation: if you add any two even numbers, you get an even number. However, the set of odd numbers is not closed under this operation: if we add, for example, $1$ and $5$, we get $6$, which is not in the set of odd numbers.

The closure of a subset $A \subset X$ under an operation $f: X^n \to X$ is the unique subset $B \subset X$ with the following three properties:

  • $A \subset B$;
  • $B$ is closed under $f$;
  • For any subset $C \subset X$ with the above two properties, $B \subset C$.

We can define this set $B$ as follows:

  • $B_0 = A$;
  • For $i \in \NN$, $B_{i+1} = B_i \cup \im(f|_{B_i})$;
  • $B = \{x \in X \mid \exists i \in \NN. x \in B_i\}$.

For example, the closure of the subset $\{3\} \subset \NN$ under addition is the set of all positive multiples of $3$.

Function Graphs

Given sets $A$ and $B$, a (binary) relation on $A$ and $B$ is a subset $R \subset A \times B$ of their product. For $a \in A$ and $b \in B$, we write $aRb$ for $(a,b) \in R$. For a function $f: A \to B$, its graph $\graph(f)$ is a relation on $A$ and $B$ defined as:

\[a \graph(f) b \iff f(a) = b\]

We can see that the graph $R$ of a function always satisfies the following two condition:

  • $\forall a \in A. \exists b \in B. aRb$;
  • $\forall a \in A, b, b' \in B. aRb \wedge aRb' \implies b = b'$.

The first condition states that every element $a \in A$ is related to some element $b \in B$, and the second property states that this element $b$ is unique. Conversely, every relation satisfying the above two properties is the graph of some function. In most formal set theories, functions aren't given as a primitive concept. Instead, functions are identified by their graphs. This allows for more flexibility with functions: you can define partial functions and multi-valued functions easily, but this comes at the cost of not being able to call functions directly. Instead, if you want the function value of a function defined by the graph $R$ given the argument $a \in A$, you'd have to use a slightly awkward $\exists b \in B. aRb \wedge \cdots$ where $\cdots$ continuous your formula.

Function graphs can be extended to graphs of $n$-ary and multi-argument functions quite easily: given a function $f: A_1 \times \cdots \times A_m \to B_1 \times \cdots \times B_n$, we can define its graph $\graph(f) \subset A_1 \times \cdots \times A_m \times B_1 \times \cdots \times B_n$ as:

\[\graph(f)(a_1,\ldots,a_m,b_1,\ldots,b_m) \iff f(a_1,\ldots,a_n) = (b_1,\ldots,b_n)\]

Given a relation $R \subset A \times B$, we write its inverse as $R^{-1}$. This is a relation on $B$ and $A$ defined by:

\[bR^{-1}a \iff aRb\]

If $R$ is the graph of a bijection, then it can be verified that $R^{-1}$ is the graph of the inverse of that bijection. Thus, this notion of an inverse relation seems to match up with the notion of an inverse function.

A relation $R \subset A \times B$ can have properties that make it look like a function or a certain kind of function. We use different notation for different kinds of relations. For example, we write $R: A \to B$ to denote $R$ satisfies the above two properties, thus signifying $R$ can be used as a usual function. Ohter properties of relations, and their notations, are:

  • $R: A \righttail B$ iff $\forall a \in A. \exists b \in B. aRb$;
  • $R: A \rightharpoonup B$ iff $\forall a \in A, b, b' \in B. aRb \wedge aRb' \implies b = b'$.

A relation $R: A \rightharpoonup B$ is sometimes called a partial function from $A$ to $B$. This can be interpreted as a graph of a function $f: S \to B$ where $S \subset A$ is a subset of $A$. In particular, $S$ is the subset $\{a \mid \exists b \in B. aRb\}$.

If we have both $R: A \righttail B$ and $R: A \rightharpoonup B$, then $R$ is just the (graph of a) function. Then, an injection can be defined as a relation $R \subset A \times B$ for which $R: A \to B$ and $R^{-1}: B \rightharpoonup A$ and a surjection is a relation $R \subset A \times B$ for which $R: A \to B$ and $R^{-1}: B \righttail A$.

Powerset

Any given set $X$ has a powerset, which consists of:

  • A set $\pow(X)$, also sometimes referred to as the powerset;
  • A function $e: X \times \pow(X) \to \Omega$.

The powerset $\pow(X)$ of $X$ can be thought of as the set of all subsets of $X$. Points in $X$ don't have additional strucutre on their own, but subsets do have extra structure: namely, the points of $X$ they contain. The function $e$ shows this extra structure of subsets. The function $e$ can be thought of as a membership relation between $X$ and $\pow(X)$: for a point $x \in X$ and a point $A \in \pow(X)$, we have that $e(x,A)$ asserts $x$ is an element of $A$.

The structure $(\pow(X),e)$, similar to products and sums of sets, as two axioms. The first concerns the construction of elements in $\pow(X)$, and the second concerns the identity relation on $\pow(X)$. These axioms are as follows:

  • For every predicate $P: X \to \Omega$, there is a corresponding element $A \in \pow(X)$ for which $\forall x \in X. e(x,A) = P(x)$;
  • For all $A,B \in \pow(X)$, if $\forall x \in X. e(x,A) = e(x,B)$, then $A = B$.

The above two axioms allow us to identify predicates on $X$ with elements of $\pow(X)$, and vice-versa. With the axiom of separation, we can also identify elements of $\pow(X)$ with subsets of $X$.

The operations $\cup$, $\cap$ and $(-)^c$ form operations on $\pow(X)$. $\cup$ and $\cap$ are binary operations and $(-)^c$ is a unary operation.

\[\begin{array}{rrcl} {\cup}: & \pow(X) \times \pow(X) & \to & \pow(X) \\ & (A,B) & \mapsto & A \cup B \\ {\cap}: & \pow(X) \times \pow(X) & \to & \pow(X) \\ & (A,B) & \mapsto & A \cap B \\ (-)^c: & \pow(X) & \to & \pow(X) \\ & A & \mapsto & A^c \end{array}\]

A structure consisting of a set and some operations on that set, sometimes satisfying a list of equations, is called an algebra. In particular, the structure $(\pow(X),\varnothing,X,{\cup},{\cap},(-)^c)$ is an example of a Boolean algebra. The symmetric difference $\triangle$ is also a binary operation on $\pow(X)$. The structure $(\pow(X),\varnothing,X,\triangle,{\cap})$ is another example of an algebra. Namely, it is a commutative ring. Besides N-algebras in the next chapter, we won't explore algebras further in this chapter.

A subset $F \subset \pow(X)$ is referred to as a (set) family of subsets of $X$. A function $G: I \to \pow(X)$ from some set $I$ is also sometimes referred to as an (indexed) family. In general, a family refers to some collection of sets or subsets. We can do large unions and large intersections on both kinds of families to get subsets of $X$. These are defined as follows:

  • $\bigcup F = \{x \in X \mid \exists A \subset X. x \in A \wedge A \in F\}$;
  • $\bigcap F = \{x \in X \mid \forall A \subset X. x \in A \wedge A \in F\}$;
  • $\bigcup_{i \in I} G(i) = \{x \in X \mid \exists i \in I. x \in G(i)\}$;
  • $\bigcap_{i \in I} G(i) = \{x \in X \mid \forall i \in I. x \in G(i)\}$.

These extend the binary $\cap$ and $\cup$ by allowing any number of sets in the union or intersection. Note that $\bigcap \varnothing = X$, as every element of $x$ is in every element of $\varnothing$ by vacuous truth.

There are many kinds of families $F \subset \pow(X)$, satisfying different properties. Some examples are topologies, $\sigma$-algebras, ideals and ultrafilters. We might see some of these kinds of families in different sections.

Though there are bijections between $\NN$ and $\NN + \NN$, and between $\NN$ and $\NN \times \NN$, Cantor showed that the same is not true for $\pow(\NN)$. His argument is called the diagonal argument, and it goes as follows:

Assume, aiming for a contradiction, we have a bijection $f: \NN \leftrightarrow \pow(\NN)$. The function $f$ is then a complete enumeration $f(0), f(1), f(2), \ldots$ of the subsets of $\NN$. We'll get to a contradiction by defining a set $A \subset \NN$ that is not on this list. First, to ensure $A$ is different from $f(0)$, we set $0 \in A$ if and only if $0 \notin f(0)$. This way, $f(0)$ and $A$ disagree, and thus $A \neq f(0)$. Second, to ensure $A$ is different from $f(1)$, we set $1 \in A$ iff $1 \notin f(1)$. By the same argument as before, $A \neq f(1)$. In general, set $n \in A$ iff $n \notin f(n)$. Thus, we have $A = \{n \in \NN \mid n \notin f(n)\}$. The set $A$ cannot be in the enumeration $f$, so the function $f$ isn't a bijection. A contradiction arises.

This argument can be extended to show that for any arbitrary set $X$, there is no bijection between $X$ and $\pow(X)$: for any supposed bijection $f$, define $A = \{x \in X \mid x \notin f(x)\}$. This way, $A$ isn't in the image of $f$, hence $f$ is not a bijection.

By iterating the powerset construction, we can get a tower of larger and larger sets $X$, $\pow(X)$, $\pow\pow(X)$, $\pow^3(X)$, $\pow^4(X)$, etc. In the section about cardinal arithmetic, we'll study these different sizes (called cardinals) further.
