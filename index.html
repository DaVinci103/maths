Sets and Functions

Sets

In mathematics, a set refers to any collection of objects. For example, there is a set of natural numbers $\NN$, a set of all people on earth, a set with no elements, and a set consisting of a triangle, the colour red and a sheep named Gregory. Sets are often denoted with curly braces $\{\cdots\}$ with the contents of the set between them. So, for example, the last set listed is denoted as $\{\triangle, \red{\bullet}, \textrm{Gregory the sheep}\}$. We'll denote sets with upper-case letters, like $A, B, C, \ldots$, and their elements with lower-case letters, such as $a, b, c, \ldots$. We use the symbol $\in$ to denote that an object is an element of a set. For example, $a \in A$ means that object $a$ is element of set $A$.

For two elements $x$ and $y$ of a set $A$, we can ask whether or not they're equal. If they are, we denote this as $x = y$. If they're not, we denote this as $x \neq y$. $=$ is called the identity relation on $A$ and has three key properties:

  • It is reflexive: for any $x \in A$, we have $x = x$;
  • It is transitive: for any three $x,y,z \in A$, if $x = y$ and $y = z$, then $x = z$;
  • It is symmetric: for two $x,y \in A$, if $x = y$, then $y = x$.

For example, in the set $\mathbb{N}$ of natural numbers, since $2+2$ is equal to $4$, by symmetry, $4$ is also equal to $2+2$. Besides elements and an identity relation, sets have no additional structure.

In the following chapters, we use a bold $\mathbf{n}$ to denote an $n$ element set and we denote its elements by $0_{\mathbf{n}}, \ldots, (n-1)_{\mathbf{n}}$. We use $\emptyset$ to denote the empty set and $! = 0_{\mathbf{1}}$ for an alternative notation of the unique element of the one element set $\mathbf{1}$. We use $\mathbb{N}$ to denote the set of natural numbers, and $0, 1, 2, 3, \ldots$ are its elements. These will be our main examples of sets.

Functions

A function $f$ from a set $A$ to a set $B$ assigns, to each element $a \in A$, some element $f(a) \in B$. Here, $a$ is called the argument of the function and $f(a)$ is the function value (at $a$).

  • This assignment must be total, meaning that $f$ has a value at every element of $A$;
  • This assignment must be unique, meaning that $f$ cannot have multiple function values at the same argument (if $a = b$, then $f(a) = f(b)$).

Two functions $f$ and $g$, both from $A$ to $B$, are equal if $f(a) = g(a)$ for all $a \in A$. This means that, besides the function values, functions do not contain any extra information, such as the exact procedure to derive those values.

We write $f: A \to B$ to denote that $f$ is a function from $A$ to $B$. The set $A$ is called the domain of the function $f$ and $B$ is called the codomain. To indicate that the function $f$ maps an object $a \in A$ to an object $b \in B$, we use the symbol $\mapsto$, as in $a \mapsto b$ (when $f$ is clear from context) or $f: a \mapsto b$. We'll often write the following:

\[\begin{array}{rrcl} f: & A & \to & B \\ & a & \mapsto & f(a) \end{array}\]

to declare a function $f$. This should make it clear if we're using a notation other than $f(a)$ to denote the function value. For example, if we write

\[\begin{array}{rrcl} (-)^*: & A & \to & A \\ & a & \mapsto & a^* \end{array}\]

then it should be clear the function value of the function $(-)^*$ applied to the parameter $a$ is denoted as $a^*$, rather than $(-)^*(a)$.

A common operation on functions is function composition. If we have functions $f$ from $A$ to $B$ and $g$ from $B$ to $C$, we can define the composite function $g \circ f$ (read: "$g$ follows $f$") from $A$ to $C$ that maps $a \in A$ to $g(f(a))$. Note that the composition $g \circ f$ is sort-of denoted right-to-left: $f$ should be applied first, then $g$. This is to make it consistent with the notation $g(f(a))$, where $g$ is also written to the left of $f$.

One (fairly obvious) property of composition is associativity: if we have three functions $f: A \to B$, $g: B \to C$ and $h: C \to D$ which we want to compose into one function $h \circ g \circ f$, it doesn't matter if we first compose $h$ with $g$, and compose the result with $f$, or if we first compose $g$ and $f$, and then compose the result of that with $h$. In both cases, we map $a$ to $h(g(f(a)))$. Thus, instead of writing $(h \circ g) \circ f$ or $h \circ (g \circ f)$, we can simply write $h \circ g \circ f$, omitting the parenthesis.

A rather trivial example of a function is the identity function. Given a set $A$, the identity function on $A$ is denoted $\id_A$. The domain and codomain of $\id_A$ are both $A$ and, for $a \in A$, $\id_A(a)$ is defined as just $a$. The identity function also is a neutral element for function composition: given a function $f: A \to B$, or $g: B \to A$, composing it with $\id_A$ gives back the same function. For $a \in A$, $(f \circ \id_A)(a) = f(\id_A(a))$ and for $b \in B$, $(\id_A \circ g)(b) = \id_B(g(b))$.

For every set $A$, there is a unique function $\mathbf{0} \to A$. Namely, the empty function (of $A$). Since $\mathbf{0}$ has no elements, the empty function doesn't need to be defined on any elements. As for functions $A \to \mathbf{0}$, there are usually none. There are no elements that the elements of $A$ can be mapped to, so no function from $A$ to $\mathbf{0}$ can be defined. The one exception is (of course) $A = \mathbf{0}$, as then we don't have elements of $A$ to worry about.
