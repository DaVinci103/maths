Sets and Functions

Sets

In mathematics, a set refers to any collection of objects. For example, there is a set of natural numbers $\NN$, a set of all people on earth, a set with no elements, and a set consisting of a triangle, the colour red and a sheep named Gregory. Sets are often denoted with curly braces $\{\cdots\}$ with the contents of the set between them. So, for example, the last set listed is denoted as $\{\triangle, \red{\bullet}, \textrm{Gregory the sheep}\}$. We'll denote sets with upper-case letters, like $A, B, C, \ldots$, and their elements with lower-case letters, such as $a, b, c, \ldots$. We use the symbol $\in$ to denote that an object is an element of a set. For example, $a \in A$ means that object $a$ is element of set $A$.

For two elements $x$ and $y$ of a set $A$, we can ask whether or not they're equal. If they are, we denote this as $x = y$. If they're not, we denote this as $x \neq y$. $=$ is called the identity relation on $A$ and has three key properties:

  • It is reflexive: for any $x \in A$, we have $x = x$;
  • It is transitive: for any three $x,y,z \in A$, if $x = y$ and $y = z$, then $x = z$;
  • It is symmetric: for two $x,y \in A$, if $x = y$, then $y = x$.

For example, in the set $\NN$ of natural numbers, since $2+2$ is equal to $4$, by symmetry, $4$ is also equal to $2+2$. Besides elements and an identity relation, sets have no additional structure.

In the following chapters, we use a bold $\mathbf{n}$ to denote an $n$ element set and we denote its elements by $0_{\mathbf{n}}, \ldots, (n-1)_{\mathbf{n}}$. We use $\emptyset$ to denote the empty set and $! = 0_{\mathbf{1}}$ for an alternative notation of the unique element of the one element set $\mathbf{1}$. We use $\mathbb{N}$ to denote the set of natural numbers, and $0, 1, 2, 3, \ldots$ are its elements. These will be our main examples of sets.

In logic, we often use quantifier symbols for statements such as "for all x, statement P holds" or "there exists a y for which statement Q holds", wherein we use a variable such as x or y to range over different values, instead of having a fixed value that the statement depends on. The symbol $\forall$ is used for the universal quantifier. The statement $\forall x.P$ means "for all x, statement P holds". The variable $x$ is placed immediately to the right of the $\forall$ statement, and the quantifier and statement $P$ are separated here with a dot. We also often bound variables to be of a certain set or type, or quantify over multiple variables at once. For example, $\forall x,y \in A, b \in B. \ldots$ means "for all $x$ and $y$ in $A$ and all $b$ in $B$, â€¦". Another common quantifier is $\exists$, which means "there exists". It is used in the same way as $\forall$, but to mean "there exists" instead of "forall". The axiom of transitivity of identity can thus be stated as $\forall x,y,z \in A. x = y \wedge y = z \implies x = z$ (the symbols $\wedge$ and $\implies$ mean "and" and "implies" respectively).

Functions

A function $f$ from a set $A$ to a set $B$ assigns, to each element $a \in A$, some element $f(a) \in B$. Here, $a$ is called the argument of the function and $f(a)$ is the function value (at $a$).

  • This assignment must be total, meaning that $f$ has a value at every element of $A$;
  • This assignment must be unique, meaning that $f$ cannot have multiple function values at the same argument (if $a = b$, then $f(a) = f(b)$).

Two functions $f$ and $g$, both from $A$ to $B$, are equal if $f(a) = g(a)$ for all $a \in A$. This means that, besides the function values, functions do not contain any extra information, such as the exact procedure to derive those values.

We write $f: A \to B$ to denote that $f$ is a function from $A$ to $B$. The set $A$ is called the domain of the function $f$ and $B$ is called the codomain. To indicate that the function $f$ maps an object $a \in A$ to an object $b \in B$, we use the symbol $\mapsto$, as in $a \mapsto b$ (when $f$ is clear from context) or $f: a \mapsto b$. We'll often write the following:

\[\begin{array}{rrcl} f: & A & \to & B \\ & a & \mapsto & f(a) \end{array}\]

to declare a function $f$. This should make it clear if we're using a notation other than $f(a)$ to denote the function value. For example, if we write

\[\begin{array}{rrcl} (-)^*: & A & \to & A \\ & a & \mapsto & a^* \end{array}\]

then it should be clear the function value of the function $(-)^*$ applied to the parameter $a$ is denoted as $a^*$, rather than $(-)^*(a)$.

A common operation on functions is function composition. If we have functions $f$ from $A$ to $B$ and $g$ from $B$ to $C$, we can define the composite function $g \circ f$ (read: "$g$ follows $f$") from $A$ to $C$ that maps $a \in A$ to $g(f(a))$. Note that the composition $g \circ f$ is sort-of denoted right-to-left: $f$ should be applied first, then $g$. This is to make it consistent with the notation $g(f(a))$, where $g$ is also written to the left of $f$.

One (fairly obvious) property of composition is associativity: if we have three functions $f: A \to B$, $g: B \to C$ and $h: C \to D$ which we want to compose into one function $h \circ g \circ f$, it doesn't matter if we first compose $h$ with $g$, and compose the result with $f$, or if we first compose $g$ and $f$, and then compose the result of that with $h$. In both cases, we map $a$ to $h(g(f(a)))$. Thus, instead of writing $(h \circ g) \circ f$ or $h \circ (g \circ f)$, we can simply write $h \circ g \circ f$, omitting the parenthesis.

A rather trivial example of a function is the identity function. Given a set $A$, the identity function on $A$ is denoted $\id_A$. The domain and codomain of $\id_A$ are both $A$ and, for $a \in A$, $\id_A(a)$ is defined as just $a$. The identity function also is a neutral element for function composition: given a function $f: A \to B$, or $g: B \to A$, composing it with $\id_A$ gives back the same function. For $a \in A$, $(f \circ \id_A)(a) = f(\id_A(a))$ and for $b \in B$, $(\id_A \circ g)(b) = \id_B(g(b))$.

For every set $A$, there is a unique function $\mathbf{0} \to A$. Namely, the empty function (of $A$). Since $\mathbf{0}$ has no elements, the empty function doesn't need to be defined on any elements. As for functions $A \to \mathbf{0}$, there are usually none. There are no elements that the elements of $A$ can be mapped to, so no function from $A$ to $\mathbf{0}$ can be defined. The one exception is (of course) $A = \mathbf{0}$, as then we don't have elements of $A$ to worry about.

For every set $A$, there also is a unque function $A \to \mathbf{1}$. If we try to define a function from $A$ to $\mathbf{1}$, we are forced to map every element of $A$ to $!$. There is a natural correspondence between functions $\mathbf{1} \to A$ and elements of $A$: a function $f: \mathbf{1} \to A$ corresponds to the element $f(!) \in A$ and, given an element $a \in A$, we can construct a function $f: \mathbf{1} \to A$ by defining $f: ! \mapsto a$.

For functions $f: A \to A$, whose domain is equal to its codomain, we can compose it with itself. We'll write repeated composition as $f^{\circ n}$ or simply $f^n$. For example, $f^{\circ 3} = f \circ f \circ f$. A function $f: A \to A$ for which $f^2 = f$ (i.e. for which applying twice is the same as applying once) is called idempotnent. If $f^2 = \id_A$, then we call it an involution. Taking an inverse or negative is a common example of an involution. We'll explore more kinds of functions (injections, surjections, bijections, permutations) in the next part. Functions that have the same domain as codomain are sometimes called operators, though this term more often refers to multi-argument functions on a single set. Endofunction is also a term that can be used to describe such a function, but this term is not used that often.

Injections, Surjections, Bijections

Some of the most important kinds of functions are injections, surjections and bijections.

For sets $A$ and $B$, an injection from $A$ to $B$ is a function $f: A \to B$ for which, for all $x,y \in A$, if $f(x) = f(y)$, then $x = y$. This is not true for functions in general: a function can map different arguments to the same value. We use the symbols $\rightarrowtail$ or $\hookrightarrow$ to indicate a function is an injection, as in $f: A \hookrightarrow B$. An injections is also sometimes called an embedding or an inclusion of $A$ in $B$, as it can be thought of as giving the elements of $A$ an interpretation as elements of $B$. This way, $A$ can be seen as a subset of $B$. The identity relation on subsets (which I won't explain right now) might be a bit unintuitive. I aim to explain more about subsets in chapter 2. It is not always possible to define an injection from one given set to another. For example, there is no injection from $\mathbf{3}$ to $\mathbf{2}$.

For sets $A$ and $B$, a surjection of $A$ onto $B$ is a function $f: A \to B$ for which $\forall b \in B. \exists a \in A. f(a) = b$ (all elements of $B$ are mapped to by some element of $A$). We use $f: A \twoheadrightarrow B$ to denote $f$ is a surjection from $A$ onto $B$. For an element $b$ of $B$, we call the subset of elements $a \in A$ that map to $b$ the fibre of $b$ under $f$, and we often denote it as $f^{-1}\{b\}$. To explain the notation: ${}^-1$ means "inverse image" and $\{b\}$ means "the set of only $b$", so $f^{-1}\{b\}$ is the "inverse image of the set of only $b$". It's clear to see, then, that a function is a surjection exactly if all its fibers are non-empty. If we think of $A$ as some ambient set and $B$ as a set of indices or labels, a surjection $f$ can then be thought of as a (labelled) partition of $A$ into non-empty subsets (namely, the fibres), each labelled, or indexed, by an element of $B$.

Another way to think of a surjection is as some sort of quotient map: the set $B$ results from the set $A$ by making the identity relation ($=$) on it weaker (making more things equal to eachother), and the surjection realizes each element of $A$ as an element of $B$. The surjections as quotients idea is dual to the injections as subsets idea, but a bit more complicated. It'll not show up for a few chapters.

Given a surjection $f: A \to B$, a section of $f$ is a function $s: B \to A$ that chooses, for each element $b \in B$, some element $a \in A$ in the fibre of $b$. Thus, $s$ is a section if $f \circ s = \id_B$ (make sure you understand this equation). It's not so difficult to prove all sections are inverses (try it yourself!). Some surjections have obvious sections. For example, if we have a function $f: \NN \to \mathbf{2}$ that maps even numbers to $0_{\mathbf{2}}$ and odd numbers to $1_{\mathbf{2}}$, then one example of a section maps $0_{\mathbf{2}}$ to the even number $4$ and $1_{\mathbf{2}}$ to the odd number $11$. For some (a bit more complicated) surjections, finding a section is less obvious. For example, where $\mathbb{R}$ denotes the real number line, and in the quotient $\mathbb{R}/\mathbb{Q}$ two reals $r_1,r_2$ are considered equal if their difference $r_1-r_2$ is rational, how to define a section of the quotient map $q: \mathbb{R} \twoheadrightarrow \mathbb{R}/\mathbb{Q}$ is less obvious.

The axiom of choice, abbreviated as AC, states that every surjection has a section. This axiom is used a lot in modern mathematics. We will not assume it by default: if we want to show a surjection has a section, an explicit construction of that section must be given. We will expand more on AC in future chapters.

A retraction of an injection is analogous to a section of a surjection: $r$ is a retraction of $f$ if $r \circ f = \id_A$. Retractions are boring and you may forget about them immediately.

A bijection from $A$ to $B$ is a function $f: A \to B$ that is both an injection and a surjection. A fibre of an injection always has at most one element, and fibres of surjections have at least one element, so fibres of bijections have exactly one element. Another way to state $f$ is bijective is by the existence of a function $g: B \to A$ for which $g \circ f = \id_A$ and $f \circ g = \id_B$. Such a $g$ is called an inverse of $f$, and is often denoted $f^{-1}$. Bijections are a one-to-one correspondence of the elements of $A$ and $B$, so it can be thought of as an equivalence of those two sets. There are different symbols that can be used to denote a function is a bijection, but we'll use $\leftrightarrow$.
